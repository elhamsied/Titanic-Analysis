# 🚢 Titanic Dataset Analysis

## 📌 Overview
This project focuses on data analysis using pandas. It includes creating a custom dataset and analyzing the Titanic dataset to extract meaningful insights.

---

## 🧩 Part 1: Custom Dataset
- Created a DataFrame using a dictionary  
- Contains 5 columns and 15 rows  
- Used custom indexing  

---

## 🚢 Part 2: Titanic Dataset

### 🔍 Exploration
- Used `head()`, `info()`, and `describe()` to understand the data  

### 🧹 Data Cleaning
- Filled missing **Age** with median  
- Filled missing **Embarked** with mode  
- Dropped **Cabin** column  
- Removed duplicate rows  

### 📊 Analysis
- Survival rate by gender  
- Survival rate by passenger class  
- Average age per class  
- Survival by age group  

### 🔎 Filtering
- Female passengers who survived  
- Children who survived  
- First-class passengers with high survival  

---

## 🧠 Key Insights
- Females were more likely to survive  
- First-class passengers had higher survival rates  
- Children were prioritized  
- Highest survival: Female passengers in 1st class
